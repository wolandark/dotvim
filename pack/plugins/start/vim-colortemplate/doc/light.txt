		  Light color scheme    by Myself

Light, a color scheme with a unique touch.

==============================================================================
Color scheme options					light-options*

						*g:light_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:light_transp_bg = 0
<
